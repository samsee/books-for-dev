# books-for-dev

개발자를 위한 추천 도서 목록을 한눈에 볼 수 있도록 정리한 정적 웹 페이지 프로젝트입니다.  
책을 주제별로 묶어 보여주고, 각 카드에서 알라딘 도서 상세 페이지로 바로 이동할 수 있습니다.

Pages URL: [https://samsee.github.io/books-for-dev/](https://samsee.github.io/books-for-dev/)

## 구성

- `index.html`: 페이지 레이아웃과 스타일
- `app.jsx`: 책 목록 렌더링과 그룹 섹션 UI
- `books-data.js`: 화면에 사용하는 도서 데이터
- `books.csv`: 원본 도서 목록 데이터
- `images/`: 책 표지 이미지
